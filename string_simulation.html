String simulator

(this page is in progress)

This is a physics-oriented, and a musical project. I gained experience of working with an audio buffer in this project. The string itself updates very quickly, as it is synchronised with the audio buffer, so there is a slow-down option that you can adjust. You can pluck the string, and make a sound. With some tricky math, I managed to make it possible to play melodies on a single string by using precisely plucked harmonics
