CPU Watcher

(this page is in progress)

I felt very curious about being able to see how the computing hardware works in real time. In order to visualise the hardware, I decided to map the layout of my CPU onto my application, and represent each CPU core as an individual tile.
On this side-by-side image, you can see how my particular model of CPU looks like, and how my application represents this CPU topology. The blue tiles are the performance cores, that support multi-threading, while the two gray tiles to the bottom - are the two blocks of 4 efficiency cores.
I have also added an experimental feature to throttle each CPU individually, in order to force the CPU to stay below a certain temperature. It works surprisingly well, and I will cover it later as I fill up this webpage with more content.
